# 🎨 Mezzotone

> Image, GIF and Video → ASCII / Unicode art --- powered by Go and
> Bubble Tea

**Mezzotone** is a terminal UI (TUI) application written in Go that
converts images or gifs into ASCII or
Unicode art.
